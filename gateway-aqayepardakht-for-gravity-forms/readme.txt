با نصب این پلاگین می توانید از خدمات درگاه آقای پرداخت برای پلاگین گرویتی فرم استفاده کنید!

== Description ==
افزونه درگاه پرداخت آقای پرداخت برای Gravity Forms

= امکانات =
 * نمایش خطاهای درگاه پرداخت
 * امکان فعال سازی حالت آزمایشی
 * پنل تنظیمات ساده و کاربرپسند
 * درج شماره کارت پرداخت کننده در بخش یادداشت ها
 * درج شماره تراکنش آقای پرداختی در بخش یادداشت ها
 * سازگار با نسخه جدید وب سرویس آقای پرداخت



== Installation ==
1. پوشه `gateway-aqayepardakht-for-gravity-forms` را در مسیر `/wp-content/plugins/` آپلود کنید
2. افزونه را از طریق منوی 'افزونه ها' در وردپرس فعال کنید
3. تنظیمات افزونه را می توانید از طریق قسمت فرم ها / آقای پرداخت انجام دهید
